# Использование справочных ресурсов

## Отслеживание подсказок команд

Ошибки при наборе команды в терминале:
- command not found
    - опечатка
    - команды не существует
    - команда не доступна текущему пользователю
- invalid option - неверный флаг(опция)
    - использовать --help для информации о флагах
- missing file operand
    - не указан операнд
- no such file or directory


## Использование помощи по встроенным командам

Встроенные команды - часть кода командного интерпретатора

Внешние команды - исполняемые файлы

`help` - получить список встроенных команд

Получить помощь по встроенной команде:
- `help имя_команды`
- `имя_команды --help`

`man builtins` - получить справку по командам, встроенным в bash

## Справочная система man

`/usr/share/man` - глобальная иерархия справочных страниц
`/usr/share/man/ru` - локализованная

Справочные страницы поделены на 8 разделов:
- 1 - исполняемые программы и команды оболочки
    - nano, ls
- 2 - системные вызовы(функции, предоставляемые ядром)
    - chmod
- 3 - библиотечные вызовы(функции библиотек)
    - cos, sin
- 4 - файлы устройств
    - mem, ram
- 5 - форматы файлов и соглашений
    - dpkg
- 6 - игры
- 7 - разное(пакеты и соглашения)
    - ascii, locale
- 8 - команды администрирования системы
    - apt, ping

`man -f объект` - показать все номера страниц и их краткое описание

`man -a объект` - открыть последовательно все страницы

`man номер_страницы объект` - открыть конкретную страницу

`man -k подстрока_поиска` - поиск по подстроке команды или подстроке описания

`apropos подстрока_поиска` - список страниц, которые содержат подстроку
